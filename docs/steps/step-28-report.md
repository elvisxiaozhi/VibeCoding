# Step 28: 长期资产管理口径 — 年化收益率 90 天阈值

## 交付内容

### 1. 年化收益率最短持有期阈值
- 新增统一阈值 `MIN_ANNUALIZED_HOLDING_DAYS = 90`
- 统一口径：有效持有期不足 90 天时不展示年化收益率
- 目的：避免短期涨跌被年化公式放大，减少对长期资产管理决策的误导

### 2. 计算层封装
- 新增 `annualizedReturnIfReady()`：
  - 单笔买入持有不足 90 天返回 `null`
  - 满 90 天后才调用原 `annualizedReturn()`
- 新增 `annualizedStartDate()`：
  - 取当前持仓买入日与可成对识别的已清仓买入日中的最早日期
  - 保持黄金类资产排除口径
  - 已清仓记录仅在 `note` 可解析出 `orig_qty` 时纳入
- 新增 `hasMinimumAnnualizedHistory()`：
  - 作为组合 / 板块 / symbol 年化 XIRR 计算前置判断
  - 未达到 90 天时返回 `false`，调用方显示 `—` 或不参与排行

### 3. Dashboard 展示口径
- 组合年化：
  - 历史汇率未加载完成时继续显示 `—`
  - 有效持有期不足 90 天时显示 `—`
- 板块年化：
  - 黄金板块继续不展示年化
  - 非黄金板块不足 90 天时不展示年化
- 年化收益率排行 Top 5：
  - 排除黄金
  - 排除不足 90 天的 symbol
  - 没有合格样本时显示「暂无持有满 90 天的年化样本」

### 4. AssetTable 展示口径
- symbol 聚合行：
  - 年化不足 90 天显示 `—`
  - 年化排序中未达标样本排到末尾
- 板块汇总：
  - 不足 90 天不显示板块年化
- 展开后的单笔买入记录：
  - 持有不足 90 天显示 `—`
  - 满 90 天后继续显示原单笔 CAGR 年化
